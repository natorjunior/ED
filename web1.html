Operador Ternário

Forma compacta de verificar condições.

let idade = 18;
let acesso = (idade >= 18) ? "Permitido" : "Negado";
console.log(acesso);

Condicional If/Else

let idade = 20;
let acesso;

if (idade >= 18) {
    acesso = "Permitido";
} else {
    acesso = "Negado";
}

console.log(acesso);

Operador Ternário

Forma compacta de if/else.


let idade = 20;
let status = (idade >= 18) ? "Permitido" : "Negado";
console.log(status);

If/Else vs. Ternário

If/Else


let idade = 20;
let acesso;

if (idade >= 18) {
    acesso = "Permitido";
} else {
    acesso = "Negado";
}
console.log(acesso);

Ternário


let idade = 20;
let status = (idade >= 18) ? "Permitido" : "Negado";
console.log(status);
