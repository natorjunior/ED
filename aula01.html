Variáveis, Operadores e Condicionais

Linguagem C

  • Base para raciocínio lógico e estruturas de dados.
  • Sintaxe simples e direta.
  • Excelentes fundamentos para programação de sistemas.

Fluxo de Programa

  • Execução linha a linha.
  • Desvios condicionais alteram o percurso.
  • Uso de variáveis para armazenar valores intermediários.

Declaração de Variáveis


#include 

int main() {
    int idade = 20;
    float peso = 70.5;
    char inicial = 'N';

    printf("Idade: %d\n", idade);
    printf("Peso: %.1f\n", peso);
    printf("Inicial: %c\n", inicial);

    return 0;
}
Atividade 1: Declare duas variáveis float (altura e peso) e exiba-as na tela.

Operadores Aritméticos

  • +, -, *, /, %
  • Realizam operações matemáticas básicas.

#include 

int main() {
    int a = 10, b = 3;
    printf("Soma: %d\n", a + b);
    printf("Subtração: %d\n", a - b);
    printf("Multiplicação: %d\n", a * b);
    printf("Divisão: %d\n", a / b);
    printf("Resto (%%): %d\n", a % b);

    return 0;
}
Atividade 2: Calcule a média de três notas (inteiras) e exiba como float.

Operadores Relacionais & Lógicos

  • Relacionais: ==, !=, >, <, >=, <=
  • Lógicos: && (E), || (OU), ! (NÃO)

Exemplo


#include 

int main() {
    int x = 5, y = 10;
    printf("(x == y) = %d\n", (x == y));
    printf("(x != y) = %d\n", (x != y));
    printf("(x < y && x > 0) = %d\n", (x < y && x > 0));
    printf("(x > y || x == 5) = %d\n", (x > y || x == 5));
    printf("!(x == 5) = %d\n", !(x == 5));

    return 0;
}
Atividade 3: 1) Declare duas variáveis inteiras e verifique se são iguais. 2) Teste combinações usando && e ||.

Estruturas Condicionais

  • if/else → desvio simples.
  • switch/case → múltiplas opções.
  • Operador Ternário → forma curta de if/else.

if/else


#include 

int main() {
    int idade = 17;

    if (idade >= 18) {
        printf("Maior de idade\n");
    } else {
        printf("Menor de idade\n");
    }

    return 0;
}
Atividade 4: Peça ao usuário que digite uma idade e informe se é Maior ou Menor de idade.

switch/case


#include 

int main() {
    int opcao = 2;

    switch(opcao) {
        case 1:
            printf("Opção 1\n");
            break;
        case 2:
            printf("Opção 2\n");
            break;
        default:
            printf("Opção inválida\n");
    }

    return 0;
}
Atividade 5: Peça um número de 1 a 4 e imprima a estação do ano correspondente.

Operador Ternário


#include 

int main() {
    int x = 10, y = 20;
    int maior = (x > y) ? x : y;

    printf("O maior valor é: %d\n", maior);

    return 0;
}
Atividade 6: Dada char sexo (M/F), use o ternário para exibir “Masculino” ou “Feminino”.
